Home Enviado Publicar Imagem Perfil Criadores

Home : é apresentada uma lista de todas as fotos enviadas para o sistema.


Publicar : local para a publicação de uma imagem. Esta terá de ter o nome do autor e um comentário.


Imagem : visualização de uma imagem selecionada , o comentário inicial, autor,data da publicação, localização e palavras-chave. É possível também verificar o nº de visualizações, votos positivos e votos negativos.


Perfil : perfil de um utilizador , as imagens enviadas por eles e os votos atribuidos.


Criadores : é possível ver os criadores deste projeto.
